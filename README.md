# Getir Challenge

## Installation
```bash
$ cd /path/to/project
$ cp .env.example .env
$ yarn install
```

## Available Scripts
In the project directory, you can run:

### `yarn lint`
Checks code for code style issues.

### `yarn start`
Runs the app. Open [http://localhost:8080](http://localhost:8080) to view it in the browser.
